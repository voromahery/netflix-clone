## Netflix clone

### What did I Used:

- Using compound components and styled component

- Using firebase for the authentication.


### How to use?

#### Run the commands below:
```
$ yarn # npm install
$ yarn build # npm run build
````

## Development

Run Webpack in watch-mode to continually compile the JavaScript as you work:

```
$ yarn watch # npm run watch
```
